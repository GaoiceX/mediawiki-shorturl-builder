% A list of reverse DNS patterns for hosts where the user likely has root access
% i.e. non-shared hosting like VPS, cloud, dedicated, and personal servers
% 
% Percent sign (%) is a comment (TeX format is misused to get syntax highlighting for comments)
% Each non-empty line is a reverse dns matching pattern
% "example.net" with no starting period will only match "example.net"
% ".example.net" will match "example.net" as well as and number of subdomains like "foo.example", "foo.bar.example.net", and so on
% # will match any number of digits, e.g. "vps#" will match "vps1" and "vps1111", but not "vpsX"
% * will match any character within  single part, i.e. "*.example.net" will match "foo.example.net" but not "foo.bar.example.net"
% (a|b) will match either "a" or "b"
